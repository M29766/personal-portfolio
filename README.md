# Manas Nandanwar — Personal Portfolio

A modern, responsive personal portfolio website built with **React.js** and **Vite**. It showcases my projects, technical skills, internship experience, education, certifications, resume, and contact information.

## Live Portfolio

[View the live portfolio](https://personal-portfolio-three-sigma-14.vercel.app/)

## Features

- Responsive design for desktop, tablet, and mobile
- Light and dark themes
- Smooth section navigation
- About and education sections
- Categorized technical skills
- Internship experience with certificate links
- Featured projects with GitHub and live-demo links
- Searchable and filterable certification section
- Resume and certificate PDF viewer
- Contact section with email integration
- Back-to-top navigation button

## Installation

Clone the repository:

```bash
git clone https://github.com/M29766/personal-portfolio.git
```

Open the project directory:

```bash
cd personal-portfolio
```

Install the dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Create a production build:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
